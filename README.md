# 页面置换模拟器

## 概述

模拟器模拟了一个简单的页式虚拟内存管理，利用`trace_path`中存放的内存访问序列计算Hit/Miss和近似比。

目前已经加入的`trace`：

- `short`：小的例子
- `matrix-inverse`：一个类似矩阵转置的访问序列

目前已经实现的算法：

- `rp_ran`：随机换出算法
- `rp_fifo`：先入先出算法
- `rp_lru`：最近最久未访问算法

## 编译

使用命令`make`编译。

## `trace`格式

首先一个数$len$，表示访存序列的长度；

接下来$len$行，每行先一个字符`R/W`，表示这个操作是读/写；接下来一个整数，表示访问的页面号。
